
📚 Library Books Sales Dashboard | Power BI & PostgreSQL
📌 Project Overview

This project is an interactive Library Books Sales Dashboard developed using Microsoft Power BI with PostgreSQL as the data source. The dashboard provides insights into book sales performance, revenue, customer purchasing trends, genre analysis, and country-wise sales through interactive visualizations.

🛠️ Tools & Technologies
Microsoft Power BI
PostgreSQL
Power Query
DAX (Data Analysis Expressions)
📊 Dashboard Features
💰 Total Revenue KPI
📚 Total Books Sold
🛒 Total Orders
📈 Average Order Sales
📖 Orders by Genre (Funnel Chart)
📊 Sales by Genre (Column Chart)
🍩 Total Revenue by Genre (Donut Chart)
🌍 Top 5 Countries by Sales
📅 Orders by Month (Line Chart)
🎯 Year Slicer (2022–2024)
📈 Key Insights
Analyzed total revenue generated from book sales.
Identified the top-performing book genres.
Found the Top 5 countries with the highest sales.
Tracked monthly order trends.
Compared revenue distribution across different genres.
Built an interactive dashboard with dynamic filtering.
🗄️ Database

The dataset was stored and managed in PostgreSQL and connected to Power BI for visualization and reporting.
